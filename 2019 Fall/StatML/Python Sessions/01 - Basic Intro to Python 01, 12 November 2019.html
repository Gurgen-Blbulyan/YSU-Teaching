Markdown Basics¶

For each cell, we have 2 Modes:

  • Command Mode (the outer, enclosing rectangle is in Blue)

  • Edit Mode (the outer, enclosing rectangle is in Green)

  • To switch from Edit Mode to Command Mode, hit Esc

  • To switch from Command Mode to Edit Mode, hit Enter

Also, we have Text/Markdown cells, and we have Code Cells.

  • When in the Command Mode, to switch from Code cell to Markdown Cell, hit m
  • When in the Command Mode, to switch from Markdown cell to Code Cell, hit y

You can use different styling options, like this: This is a very basic intro to Jupyter, Markdown and Python Language.

Also, you can include HTML code in the Markdown, like the above color codes or the header code. Or something like

Tralala 1 Tralala 2
Tralala 3 Tralala 4
This is a Table

Even you can write here Հայերեն $\ddot{\smile}$

Another nice thing is that you can include LaTeX formulas into your Markdown code, like: $$ \int_0^1 \left(\sum_{n=1}^\infty f_n(x)\right)\, dx = 1 $$

Or, say, if $\mathcal{X}\subset\mathbb{R}$, $\pi<\gamma$ and $X\sim\chi^2(n)$, then $Z\sim\mathcal{N}(0,1)$, assuming that $$ \frac{\overline{X}-\mu}{\sigma/\sqrt{n}} \to 12345 \qquad as \qquad n\to+\infty. $$
